# CalorieAI 全面修复与优化规格文档

## 变更ID

comprehensive-fix-optimization

## 变更原因

当前应用存在多个严重问题需要修复：

1. AI助手三种形态UI需要重构优化
2. AI对话存在并发提交问题，需要防止重复提交
3. AI对话无法使用（DNS解析失败），需要修复LongCat API调用
4. 缺少流式对话效果（打字机效果）
5. 热力图数据展示不正确
6. 月度总结默认显示逻辑需要调整
7. 应用启动缓慢
8. 模型配置界面存在模拟配置需要删除
9. 首次打开一直转圈（新手引导加载问题）
10. 需要审查数据备份与恢复功能
11. 需要统计文件并分析优化空间
12. 设置界面需要添加模型调用记录

## 变更内容

### 1. AI助手UI重构

**BREAKING**: 重构三种形态UI（悬浮球、迷你窗口、全屏）

#### 1.1 悬浮球形态优化

* 尺寸：56dp主按钮 + 64dp脉冲光环

* 动效：脉冲呼吸动画（1.0-1.2倍缩放）

* 颜色：主题色渐变背景

* 交互：按压缩放反馈

#### 1.2 迷你窗口形态优化

* 尺寸：360dp宽 × 520dp高（最大）

* 圆角：24dp

* 毛玻璃背景效果

* 头部：AI头像 + 状态指示 + 操作按钮

* 内容：消息列表 / 快捷操作

* 底部：玻璃风格输入框

#### 1.3 全屏形态优化

* 复用迷你窗口样式

* 扩展为全屏布局

* 添加历史会话侧边栏

  <br />

### 2. AI对话并发控制

**BREAKING**: 添加对话提交锁定机制

* 发送消息时禁用输入框和发送按钮

* 显示加载状态指示器

* 等待AI回复完成后才能继续发送

* 添加取消请求功能（可选）

### 3. LongCat API修复

**BREAKING**: 修复API调用问题

#### 3.1 API URL修复

* 当前URL：`https://api.longcat.chat/openai/v1/chat/completions`

* 问题：返回405 Method Not Allowed

* 解决方案：

  * 联网搜索或者查看之前git提交的源码进行修复

    * 检查正确的API endpoint格式

    * 可能需要使用 `https://api.longcat.chat/v1/chat/completions`

    * 或使用 `https://longcat.chat/api/v1/chat/completions`

#### 3.2 网络错误处理

* 添加DNS解析失败提示

* 添加网络超时处理

* 添加重试机制

### 4. 流式对话效果

**新增功能**: 实现打字机效果

#### 4.1 流式接收

* 使用SSE（Server-Sent Events）接收流式数据

* 解析 `data: {...}` 格式的流式响应

* 处理 `[DONE]` 结束标记

#### 4.2 打字机渲染

* 逐字显示动画（30-50ms/字符）

* 光标闪烁效果

* 支持Markdown实时渲染

#### 4.3 AI回复可读性优化（重要）

**文本格式优化：**
* 段落间距：1.5倍行高，段落间增加8dp间距
* 字体大小：正文14sp，标题16sp，小标题15sp
* 字体粗细：标题加粗，正文常规
* 文字颜色：深色模式下使用高对比度颜色（#E8DEF8），浅色模式使用（#1D1B20）

**Markdown渲染优化：**
* 标题：H1/H2/H3 使用不同字号和粗细，添加底部边框线
* 列表：有序列表使用数字编号，无序列表使用圆点，缩进层级清晰
* 代码块：深色背景 + 等宽字体 + 语法高亮 + 圆角边框
* 引用块：左侧竖线 + 斜体文字 + 浅色背景
* 表格：表头加粗 + 斑马纹背景 + 边框线
* 链接：主题色 + 下划线 + 点击跳转

**内容结构优化：**
* 自动识别并格式化营养数据（如：热量、蛋白质等）
* 数字和单位之间添加空格（如：1500 kcal → 1500 kcal）
* 重要数据使用粗体或颜色高亮
* 建议类内容使用图标前缀（💡、⚠️、✅）
* 分步骤说明使用编号列表

**交互增强：**
* 长按文本可复制
* 点击链接可跳转
* 代码块可一键复制
* 表格可横向滚动

### 5. 热力图数据修复

## 技术要求

### 性能要求

* 启动时间 < 2秒

* UI渲染帧率 > 60fps

* 流式对话延迟 < 100ms首字

### 兼容性要求

* Android 8.0+

* 支持浅色/深色模式

* 适配不同屏幕尺寸

### 安全要求

* API密钥加密存储

* 备份数据加密（可选）

* 敏感信息脱敏记录
