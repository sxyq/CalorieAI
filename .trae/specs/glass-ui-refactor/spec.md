# CalorieAI Glass UI 重构规格文档

## 变更ID
glass-ui-refactor

## 变更原因
当前应用界面需要全面升级，采用 Material Design 3 + Glass 毛玻璃设计风格，提升视觉品质和用户体验。具体需求：
1. 导航栏需要毛玻璃视觉效果，增强现代感
2. 颜色系统需要适配浅色/深色模式
3. 动效设计需要更加流畅自然
4. 所有界面需要统一的视觉风格

## 变更内容

### 1. 毛玻璃视觉效果系统
**BREAKING**: 全面替换原有背景样式

#### 1.1 导航栏毛玻璃背景
- 尺寸: 高度80dp，宽度100%
- 圆角: 0dp（直角）
- 布局: 水平等分布局
- 毛玻璃属性:
  - 背景色: #F3EDF7（浅色）/#211F26（深色）
  - 透明度: 95%（轻微透出底层内容）
  - 模糊: 20-30px高斯模糊
  - 边框: 1px solid rgba(255,255,255,0.1)（浅色）| rgba(0,0,0,0.1)（深色）
  - 顶部高光: 1px渐变（rgba(255,255,255,0.2) → transparent）

#### 1.2 胶囊指示器
- 尺寸: 64dp × 32dp
- 圆角: 50%（胶囊形）
- 毛玻璃叠加: 10-15px模糊
- 内阴影: 底部1px rgba(0,0,0,0.1)
- 位移: 选中时轻微上浮2dp

#### 1.3 卡片毛玻璃效果
- 背景透明度: 85-90%
- 模糊: 15-20px
- 边框: 1px半透明白色/黑色
- 圆角: 16-24dp

### 3. 动效设计系统

#### 3.1 动画时长与曲线
| 效果 | 时长 | 缓动曲线 |
|------|------|----------|
| 指示器出现/消失 | 300ms | cubic-bezier(0.4, 0.0, 0.2, 1) |
| 指示器缩放 | 250ms | cubic-bezier(0.34, 1.56, 0.64, 1) |
| 图标/文字变色 | 200ms | cubic-bezier(0.4, 0.0, 0.2, 1) |
| 涟漪扩散 | 400ms | cubic-bezier(0.4, 0.0, 0.2, 1) |
| 图标弹跳 | 150ms | cubic-bezier(0.34, 1.56, 0.64, 1) |

#### 3.2 指示器动效细节
- 出现: 从中心缩放（0.8→1）配合淡入（opacity: 0→1）
- 可选: 圆角渐变（100%→50%），从圆形平滑过渡到胶囊形
- 光泽: 选中时顶部微妙的流动高光效果

#### 3.3 涟漪效果
- 触发: 以触摸点为中心向外扩散
- 范围: 覆盖整个导航项
- 透明度: 按下12%、聚焦10%、悬停8%
- 混合模式: overlay或screen增强发光感

#### 3.4 图标微动效
- 选中时: 放大1.1倍，50ms弹性回弹
- 切换时: 轻微缩放跳动（1→1.1→1）
- 悬停时: 上浮2px（桌面端）

### 4. 界面重构范围

#### 4.1 引导流程界面（6个）
- OnboardingScreen1.kt - 基本信息
- OnboardingScreen2.kt - 身体数据
- OnboardingScreen3.kt - 生活习惯
- OnboardingScreen4.kt - 目标设定
- OnboardingScreen5.kt - 目标细化
- OnboardingScreen6.kt - 完成确认

#### 4.2 主界面
- 概览页面（OverviewScreen）
- 功能入口页面（FunctionsScreen）
- 身体档案页面（BodyProfileScreen）
- 我的界面（MyScreen）

#### 4.3 组件重构
- 底部导航栏（BottomNavBar）
- 热力图组件（HeatmapCalendar）
- 月度总结卡片（MonthlySummaryCard）
- Markdown渲染组件（MarkdownText）
- 打字机效果组件（TypewriterText）
- 聊天历史管理（ChatHistoryManager）
- AI头像动画（AIAvatarAnimations）
- 增强体重图表（EnhancedWeightChart）

### 5. 设计要点
1. 导航栏毛玻璃：95%透明+20-30px模糊+顶部高光
2. 胶囊指示器：带模糊叠加+底部内阴影
3. 指示器动画：缩放+淡入，弹性曲线增强动感
4. 图标微动效：选中时1.1倍放大+弹性回弹
5. 涟漪效果：触摸点扩散+overlay混合模式增强发光
6. 卡片设计：统一圆角16-24dp，毛玻璃背景
7. 文字层级：标题/正文/辅助文字清晰区分

## 影响范围

### 受影响模块
- 所有UI界面（引导流程、主界面、组件）
- 主题配置（Theme.kt, Color.kt）
- 底部导航栏
- 所有卡片组件

### 修改文件
- Theme.kt - 更新颜色系统
- Color.kt - 定义Glass风格颜色
- BottomNavBar.kt - 毛玻璃导航栏
- 所有Screen文件 - 应用Glass风格
- 所有组件文件 - 应用Glass风格

## 技术要求

### 性能要求
- 模糊效果使用RenderScript或GPU加速
- 动画帧率保持60fps以上
- 低端设备降级为纯色背景

### 兼容性要求
- Android 8.0+支持
- 支持动态主题切换
- 适配不同屏幕尺寸
