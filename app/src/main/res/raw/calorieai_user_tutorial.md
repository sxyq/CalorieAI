# CalorieAI 用户操作教程

**欢迎使用 CalorieAI - 您的智能健康管理助手**

---

## 📱 应用概览

CalorieAI 是一款功能全面的智能健康管理应用，主要功能包括：

| 功能模块 | 描述 |
|---------|------|
| 🍽️ 饮食记录 | 通过AI拍照识别或手动录入食物 |
| 💧 饮水追踪 | 记录每日饮水量，支持快捷添加 |
| 🏃 运动记录 | 记录各类运动消耗的卡路里 |
| ⚖️ 体重管理 | 追踪体重变化趋势 |
| 📊 数据统计 | 多维度数据可视化分析 |
| 🤖 AI助手 | 智能问答与个性化建议 |

---

## 第一章：初次使用设置

### 1.1 欢迎页面

首次打开应用，您将看到欢迎页面。

> **【图片插入位置 1-1】**
> **此处插入：OnboardingScreen1 截图**
> **图片说明：欢迎页面 - 显示 "让我们认识您" 标题，包含性别选择（男/女/其他）、出生年月选择器、身高体重输入框**

**操作步骤：**

1. **选择性别**
   - 点击选择：👨 男性、👩 女性 或 😊 其他
   - 选中项会高亮显示

2. **设置出生年月**
   - 点击【年份】选择出生年（范围：当前年-80 至 当前年-15）
   - 点击【月份】选择出生月（1-12月）
   - 选择后自动保存

3. **输入身高体重**
   - 身高：点击 +/- 按钮调整，范围 100-220cm
   - 体重：点击 +/- 按钮调整，范围 30-200kg
   - 也可以直接点击数值手动输入

4. 点击【下一步】继续

---

### 1.2 设置健康目标

接下来您需要设置个人的健康目标。

> **【图片插入位置 1-2】**
> **此处插入：OnboardingScreen2 截图**
> **图片说明：目标设置页面 - 显示目标类型选择（减重/维持/增重），以及目标体重输入**

**操作步骤：**

1. **选择目标类型**
   - 🔥 减重：科学减重，健康瘦身
   - ⚖️ 维持：保持当前体重
   - 💪 增重：健康增肌增重

2. **设置目标体重**
   - 应用会根据您的当前体重给出建议范围
   - 点击数值进行调整

3. **选择达成策略**（如选择减重）
   - 温和策略：每周减0.25kg，舒适可控
   - 标准策略：每周减0.5kg，推荐选择
   - 积极策略：每周减0.75kg，快速见效
   - 激进策略：每周减1kg，需严格执行

4. 点击【下一步】继续

---

### 1.3 设置活动水平

根据您的日常活动量进行选择。

> **【图片插入位置 1-3】**
> **此处插入：OnboardingScreen3 截图**
> **图片说明：活动水平设置 - 显示久坐/轻度/中度/高度/极高等选项**

**活动水平说明：**

| 水平 | 描述 | 适合人群 |
|------|------|---------|
| 🪑 久坐不动 | 几乎不运动，主要坐着工作 | 办公室白领 |
| 🚶 轻度活动 | 每周1-3天轻度运动 | 偶尔散步者 |
| 🏃 中度活动 | 每周3-5天中度运动 | 规律健身者 |
| 🏋️ 高度活动 | 每周6-7天高强度运动 | 运动爱好者 |
| 🏆 极高活动 | 专业运动员或体力劳动者 | 运动员/重体力工作者 |

点击适合您的选项后，点击【下一步】。

### 1.4 饮食偏好设置

设置您的饮食偏好以获得更精准的建议。

> **【图片插入位置 1-4】**

## 第三章：饮食记录

### 3.1 AI拍照识别

使用AI拍照功能快速识别食物。

> **【图片插入位置 3-1】**
> **此处插入：CameraScreen 截图**
> **图片说明：相机界面 - 显示相机预览、拍照按钮、相册选择、闪光灯开关**

**操作步骤：**

1. 点击【AI拍照识别】进入相机界面
2. 将食物放在镜头中央，确保光线充足
3. 点击底部圆形按钮拍照
4. 或点击相册图标从相册选择已有照片

> **【图片插入位置 3-2】**
> **此处插入：PhotoAnalysisScreen 截图**
> **图片说明：照片分析结果 - 显示识别的食物名称、预估卡路里、营养成分（蛋白质/碳水/脂肪）、置信度指示器**

**查看分析结果：**
- 食物名称和数量
- 总卡路里
- 营养成分：蛋白质、碳水化合物、脂肪
- AI置信度：高/中/低
- 编辑选项：可修改识别结果

点击【确认添加】保存记录，或点击【重新识别】更换照片。

---

### 3.2 手动添加食物

如果AI识别不准确，或您想手动记录。

> **【图片插入位置 3-3】**
> **此处插入：ManualAddScreen 截图**
> **图片说明：手动添加界面 - 显示食物名称输入框、卡路里输入、餐食类型选择（早餐/午餐/晚餐/加餐）、数量设置**

**操作步骤：**

1. **搜索食物**
   - 在搜索框输入食物名称
   - 或选择常用食物快捷按钮

2. **选择餐食类型**
   - 🌅 早餐 (7:00)
   - 🌞 午餐 (12:00)
   - 🌆 晚餐 (18:00)

3. **设置数量**
   - 选择单位：克、份、碗、个等
   - 输入具体数量

4. **确认信息**
   - 自动计算总卡路里
   - 点击【添加】保存

## 第十章：数据管理

### 10.1 数据备份

保护您的健康数据。

> **【图片插入位置 10-1】**
> **此处插入：BackupSettingsScreen 截图**
> **图片说明：备份设置页面 - 显示备份/恢复按钮、上次备份时间、自动备份开关**

**当前支持的数据范围（本地与云端一致）：**
- 饮食记录、运动记录、体重记录、饮水记录
- 收藏菜谱
- AI对话历史、API调用日志
- 用户设置
- AI配置（可选）

**本地备份：**
- 📤 创建备份：导出为 JSON 文件
- 📥 恢复备份：从 JSON 文件恢复

**云备份（WebDAV）：**
- 配置项：服务器地址、目录、文件名、用户名、密码/应用专用密码
- ☁️ 上传云备份：把当前完整数据上传到 WebDAV
- ☁️ 从云端恢复：先读取云端备份信息，再进入恢复确认

**恢复方式（恢复前可选）：**
- `全量覆盖`：清空当前数据后恢复
- `合并导入`：保留当前数据并导入；同ID数据覆盖更新

**恢复预览（新增）：**
- 在点击“恢复”前，会显示每类数据的影响范围
- 全量覆盖模式显示：`备份X，将清空Y`
- 合并导入模式显示：`新增X，覆盖Y`

## 🆘 常见问题

### Q1: 为什么AI识别不准确？
**A:** 请确保：
- 光线充足
- 食物清晰可见
- 避免反光和阴影
- 尽量拍摄完整食物

### Q2: 如何修改错误的记录？
**A:** 长按首页的任意记录项，选择编辑或删除。

### Q3: 卡路里计算准确吗？
**A:** 基于标准食物数据库和AI算法估算，仅供参考。具体数值因食材、烹饪方式而异。

### Q4: 如何设置多个提醒？
**A:** 进入【设置】→【通知设置】，为每种提醒类型单独设置时间。

### Q5: 数据会丢失吗？
**A:** 数据默认存储在本地。建议定期做本地备份，并可使用 WebDAV 云备份做异地保存。恢复前可查看“恢复预览”确认影响范围。

### Q6: 全量覆盖和合并导入有什么区别？
**A:** 全量覆盖会先清空当前数据再恢复；合并导入会保留现有数据，只新增或覆盖同ID的数据。建议先看恢复预览再操作。

### Q7: 云备份支持哪些服务？
**A:** 只要支持标准 WebDAV 的服务都可用，例如坚果云、Nextcloud、群晖 WebDAV 等。
